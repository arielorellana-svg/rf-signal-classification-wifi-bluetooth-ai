# Models

The trained MATLAB model file is not included in this repository because it may be large.

To reproduce the model, run these scripts from the `matlab/` folder:

1. `01_generate_dataset_wifi_bluetooth_v3.m`
2. `02_train_cnn_wifi_bluetooth_v3.m`

The final trained model is saved locally as:

`models/cnn_wifi_bluetooth_v3_domain_randomized.mat`
