# Models

This folder contains the trained MATLAB models used to reproduce the reported results.

## Included models

- `cnn_wifi_bluetooth_v3_domain_randomized.mat`  
  Custom CNN baseline trained from scratch.

- `resnet18_transfer_learning_wifi_bluetooth.mat`  
  ResNet-18 transfer-learning model fine-tuned for the six RF spectrogram classes.

These files allow reviewers to verify the results without retraining the networks.
