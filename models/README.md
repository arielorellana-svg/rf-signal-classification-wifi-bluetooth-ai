# Models

The trained MATLAB model file is expected at:

`models/cnn_wifi_bluetooth_v3_domain_randomized.mat`

This file is required to verify the reported results without retraining.

If the model file is not present, it can be regenerated by running:

1. `matlab/step01_generate_dataset_wifi_bluetooth.m`
2. `matlab/step02_train_cnn_wifi_bluetooth.m`

After training, the model is saved locally as:

`models/cnn_wifi_bluetooth_v3_domain_randomized.mat`

Note: The pretrained model should be included in the repository or tracked using Git LFS if the file is large.
