# Data

Large generated datasets are not included in this repository.

The synthetic datasets can be regenerated using the MATLAB scripts in the `matlab/` folder.

Main datasets used in the project:

- `spectrograms_v3_domain_randomized`: 18,000 synthetic spectrograms.
- `blind_test_v2_final`: 6,000 independent blind test spectrograms.

Public NIST I/Q recordings will be used for real-world validation.
