# Data

Large generated datasets are not included in this repository because of size.

The datasets can be regenerated using the MATLAB scripts in the `matlab/` folder.

Main datasets used in this project:

- `data/spectrograms_v3_domain_randomized`: 18,000 synthetic spectrograms for training, validation, and internal testing.
- `data/blind_test_v2_final`: 6,000 independent blind-test spectrograms.
- `data_sdr/spectrograms`: 3,600 SDR-style spectrogram images used for additional validation.
